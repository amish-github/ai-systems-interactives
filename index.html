AI Systems Engineering — Lesson Interactives

87 self-contained interactive visuals for Sprints 1 to 6, built to the TripleTen V0 Interactive Visuals Guide.

Each interactive is a standalone page. To embed one in a lesson (Notion, the LMS, anywhere), use an iframe pointing at its URL, for example:
<iframe src="https://amish-github.github.io/REPO/1.1.03-sync-vs-async.html" width="100%" height="640" style="border:0"></iframe>
Click Preview on any card to see it inline. Before publishing: swap placeholder icons for Untitled UI icons and add the PostHog snippet.
