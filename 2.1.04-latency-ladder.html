Latency Ladder — click to explore

Read from memory
ns → µs
Data already lives inside your process — no hardware I/O happens at all. For almost any capacity reasoning, this is effectively free. It sets your baseline.
Read from local SSD
0.1 ms → few ms
Data moves from storage hardware into memory. No network is involved, but the hardware access itself takes time — typically hundreds to thousands of times slower than a memory read.
Network, same data center
<1 ms → few ms
Even machines that are physically close require a full network round-trip: the OS, the NIC, the switch, and back again. Distance is short but the stack overhead is real and unavoidable.
Network, cross region
10s → 100s ms
Data travels hundreds or thousands of miles. The speed of light sets a floor no engineering can remove. A cross-region call can be tens of thousands of times slower than a memory read.

Bars use a logarithmic scale — each step is roughly 1,000× slower than the one before it.
